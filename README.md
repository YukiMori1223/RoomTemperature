# RoomTemperature
 1年分の自宅の室温と気温の比較

![sample](img2.png) 
## 特徴
- Switchbotを用いた室温データの利用
    - IoTシステムの[switchbot](https://www.switchbot.jp/)では、気温・室温を測定する製品がある。
    - これを一年ほど利用し、気温データが出力できたので、どうせならと東京都の気温データと比較した。
